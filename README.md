# demo-cordinates-api

Node JS and MongoDB app that geo-fences and geo-filters

## Requirements
* Node JS
* MongoDB - Create one at [mLab](https://mlab.com)

## Getting Started
* Clone this repo
* `cd demo-cordinates-api` and run `npm install`
* `cp .env_sample .env`
* Fill out the `.env` file with details from your `mLab` MongoDB
* run `npm start`
